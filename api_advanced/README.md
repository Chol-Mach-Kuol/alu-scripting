# API Advanced

Python scripts that query the Reddit API using recursive functions and pagination.

## Files

| File | Description |
|---|---|
| `0-subs.py` | Returns number of subscribers for a subreddit |
| `1-top_ten.py` | Prints titles of top 10 hot posts for a subreddit |
| `2-recurse.py` | Recursively returns all hot article titles for a subreddit |
| `3-count.py` | Recursively counts and prints sorted keyword occurrences in hot titles |

## Usage

```bash
python3 0-main.py <subreddit>
python3 1-main.py <subreddit>
python3 2-main.py <subreddit>
python3 3-main.py <subreddit> '<keywords>'
```
